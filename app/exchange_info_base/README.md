# Exchange Info Base

Локальная “инфобаза” по биржам.

Цель:
- хранить проверенные способы получения цен;
- хранить типы самих цен и их смысл;
- хранить типы аккаунтов и режимы аккаунтов;
- хранить типы рынков/контрактов в нативных названиях биржи;
- хранить проверенные account/trade/market endpoints;
- дальше использовать это как единый справочник в коде.

Принцип:
- одна папка на одну биржу;
- внутри — только проверенные официальные endpoints и нативные термины;
- сначала покрываем то, что реально нужно проекту, но структура уже рассчитана на рост.

Сейчас заведено:
- `binance`
- `bybit`
- `bitget`
- `okx`
- `mexc`

Основные точки входа:
- `app/exchange_info_base/registry.py`
- `app/exchange_info_base/common.py`
- `app/exchange_info_base/exchanges/*/reference.py`

В базе по каждой бирже лежат:
- `account_types` — режимы и типы аккаунтов;
- `market_types` — типы рынков и контрактов;
- `price_types` — виды цен, которые встречаются в API;
- `endpoints` — проверенные REST endpoints с exact path, типом auth, способом адресации (`bulk`, `symbol`, `instType`, и т.п.), нужными полями ответа и официальными source URL.
